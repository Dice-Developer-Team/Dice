前言
-----------

   欢迎使用Dice!掷骰机器人,这是一篇简单的说明文档，但其中包含了有关用户协议和申请方法等重要信息，请在使用机器人前仔细阅读。


欢迎
------------

   Dice!是一款基于酷Q, 采用 **C++11** 编写的掷骰插件，编译环境为 **VS2017** 。

   如果您不了解酷Q是什么，请访问: `酷Q官网 <http://cqp.cc/>`_  

   如果您想要查看源代码: 请访问: `Github Page <http://kokona.tech/>`_ 以及 `Github <http://github.com/w4123/Dice/>`_  

   如果您想要直接使用掷骰服务: 请参考 `公开掷骰机器人列表 <./DiceList.html>`_

   如果您想要了解更多信息，请加入QQ群941980833或624807593(已满)


关于云掷骰服务的重要通知
------------------------------

    1.使用官方公用掷骰机器人则代表您同意了 `Dice!云掷骰服务用户协议 <./License.html>`_ ，如果您在使用公用掷骰机器人时出现任何违反上述用户协议的行为， **我们有权直接将您封禁!** 

    2.如果您想要在群中使用官方掷骰机器人，需要在将机器人拉入群后,按照 `申请步骤 <./Apply.html>`_ 申请后方可使用。
